android Jenkins自动化打包

简介

Jenkins作为一个开源的持续集成工具，不仅可以用来进行Android打包，也可以用来进行iOS打包、NodeJs打包、Jave服务打包等。官方地址为:https://jenkins.io/。

作用：打包和开发工作分离，规范打包流程，减少开发工作量，减少出错率。

本次主要是记录mac系统上的操作流程

需要安装的环境

JDK、Android SDK、Jenkins、Gradle（安装studio可以使用studio目录下的gradle）、Git。

Jenkins安装和配置

  • 基础安装

    方法一、下载安装可以到Jenki官网（http://jenkins.io）下载Mac版本的安装包

    方法二、使用brew下载安装Jenkins 如下：

    命令行执行：＃brew install jenkins

    1. 启动 安装成功后 运行命令：＃jenkins 启动之后打开浏览器，在地址栏中输入localhost:8080就可以进入jenkins服务了
    2. 插件的安装
      进入后，会让你选择推荐插件安装还是自定义插件安装（我们直接点击推荐安装）
    3. 注册账户登陆

  • 环境配置

    环境变量的配置 （系统管理 -> 系统设置 -> 环境变量）

    键：ANDROID_HOME
    值：sdk的目录

    系统变量（系统管理 -> 全局工具配置）

    1.JDK

    别名=随意。

    JAVA_HOME=即环境变量中JAVA_HOME中的内容(/Library/Java/JavaVirtualMachines/ jdk1.8.0_161.jdk/Contents/Home)

    2.Gradle
    别名=随意
    GRADLE_HOME= gradle的安装目录路径xx/gradle\gradle-2.14.1(/Applications/Android Studio.app/Contents/gradle/gradle-3.2)

    3.Git

    Name=别名

    Path to Git executable = git.exe文件的全路径（/usr/local/bin/git）
